# Live Sports Intelligence

A real-time sports video intelligence system that detects key plays from live or pseudo-live game footage, generates highlight clips, and exposes events through backend APIs and a live dashboard.

## Tech Stack
- Spring Boot
- Python
- Kafka
- OpenCV
- FFmpeg
- EasyOCR
- PostgreSQL
- Redis
- React
- Docker Compose
- Prometheus
- Grafana

## Planned Services
- backend-api
- video-ingestion
- detection-service
- highlight-service
- frontend
- infra
